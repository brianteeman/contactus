What's new

Support for Joomla 6. We now officially support Joomla 6. Please note that our software's compatibility has been validated with Joomla 6.0.0 Release Candidate 3, the final pre-release version of Joomla! 6.0.

PostgreSQL support (experimental). You can now install Akeeba DataCompliance on Joomla sites regardless of whether they use a MySQL, MariaDB, or a PostgreSQL database.

Preliminary PHP 8.5 compatibility. We continue to work on compatibility with the upcoming PHP 8.5 version. Please note that PHP 8.5 is still in a testing (Release Candidate) phase. As such, support for it is currently marked as preliminary. The compatibility goal for new PHP versions is to not have any errors and warnings in our software running under that version. Errors and warnings coming from the CMS and/or third-party software are outside our control. Our software may result in deprecation notices being emitted. While we do try to address them, doing so is not always possible as we have to support older PHP versions which may lack support for the newer PHP language features necessary to suppress these deprecation notices.

Bug fixes and minor improvements. Please take a look at the CHANGELOG below.
